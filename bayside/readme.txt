About various files and directories                                             |
***********************************                                             |
                                                                                |
All the content of the website is in the 'Sites/bayside' folder.                |
                                                                                |
--> index.html                                                                  |
        This is the homepage of the website. This would be displayed by default |
        when run on the server.                                                 |
                                                                                |
--> sights.html                                                                 |
        This is just another page on the website.                               |
                                                                                |
--> menu.js                                                                     |
        This is a javascript file in the 'js' folder.                           |  
                                                                                |
--> responsive.css                                                              |
        A css file, which governs the styling if the webpage. It is in the      |   
        styles folder.                                                          |
                                                                                |
--> images                                                                      |
        The images folder contains all the images that were used for the        |
        website.                                                                |
                                                                                |
--> part[#]                                                                     |
        All these directories contain the step by step development of the       |
        website.                                                                |
